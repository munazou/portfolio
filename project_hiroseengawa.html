広瀬川ストリートファニチャー展

群馬県前橋市 / インスタレーション / 2024
Role: Concept / Design / Fabrication / Photo Documentation

CONCEPT

前橋の中心を流れる「広瀬川」を舞台に、まちと人をつなぐ仮設家具を展示するアートイベント。 川沿いの通路や階段など、普段見過ごされる空間を“居場所”として再発見することを目的に、 家具を通じた身体的な体験を提案した。

OUTLINE

・会場：前橋市 広瀬川河畔
・テーマ：「川辺の居方をデザインする」
・出展年：2024（第5回）
・主催：広瀬川アーツプロジェクト実行委員会

NOTE

広瀬川沿いの階段を“椅子”として捉え直し、人が腰をかける、語らう、眺めるといった行為を引き出す装置として設計。 既存環境のリズムや段差を活かしながら、仮設的な構造を最小限の介入で構築した。 イベント当日は多くの通行者が立ち止まり、川と人の関係を感じ直す時間となった。
