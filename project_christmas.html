Christmas Garden

クリスマスマーケット / 群馬県庁前広場 / 20XX
Role: Layout Design, Spatial Direction, Installation

CONCEPT

県庁前の大きな広場を、冬の一夜限りの「庭」に変える試み。 屋台やイルミネーションだけでなく、人が立ち止まりたくなる 密度の違うエリアを点在させることで、広い外部空間に いくつかの居場所のリズムをつくり出している。

OUTLINE

・場所：群馬県庁前広場
・内容：マーケット／ライブステージ／ワークショップ など
・担当：全体レイアウト計画 / 動線計画 / 雰囲気づくりの提案

NOTE

昼と夜で表情が大きく変わるイベントだったため、 日中のにぎわいと夜の光の見え方の両方を意識して配置を組んだ。 写真の記録からも、参加者の滞在時間や立ち止まり方を読み取り、 今後の屋外イベントの設計にフィードバックしている。
