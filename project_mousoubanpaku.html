妄想万博（Mousou Banpaku）

前橋市 / アートインスタレーション / 2023
Role: Concept / Design / Model / Exhibition

CONCEPT

「もしも自分の中に世界があるとしたら？」という妄想から始まった小さな万博。 現実と空想のあいだにある“装置”をテーマに、家具・建築・インスタレーションを横断しながら、 日常に潜む「発見の芽」を立ち上げる試み。 会場内では、観客自身の想像を引き出すための模型・映像・音が展示された。

OUTLINE

・会場：群馬県 前橋市 広瀬川周辺
・形式：アートインスタレーション展示（共同制作）
・内容：構想・模型制作・展示構成・記録撮影
・担当：コンセプト立案／展示デザイン／模型ディレクション

NOTE

建築を“展示”として見せることで、用途や機能に縛られない空間の可能性を探った。 「椅子になりきれない椅子」「意味を脱いだ道具」など、 形態と思考の中間にある存在を、遊び心と哲学のあいだで再構築している。
