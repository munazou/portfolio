01production Event Space

代々木第一体育館 / イベントスペース / 2022
Role: Exhibition Design, Graphic Coordination, Layout Planning

CONCEPT

大規模な競技場の一角に、一時的に立ち上がるイベントブースを計画。 既存空間のスケール感を活かしながら、ブランドメッセージが遠くからでも 立ち上がって見えるよう、立体的なグラフィックと映像演出を組み合わせた。

OUTLINE

・会場：国立代々木競技場 第一体育館
・用途：イベントブース / プロモーションスペース
・期間：会期中のみの仮設インスタレーション
・担当：レイアウト検討 / 什器デザイン / サイン計画 / パース制作 など

NOTE

アリーナ全体の動線の中で、ブースがどのように見えるかを意識しながら ボリューム構成と照明計画を行った。来場者の撮影行動も想定し、 写真や動画に残ったときの画角やロゴの見え方を繰り返し検証している。 一時的な場であっても、記憶に残る「画」として成立することを重視した。
