道の駅まえばし赤城 / カフェ空間 / 2023
Role: Design / Coordination（インターンとして担当）

CONCEPT

道の駅という日常と観光のあいだにある場所に、地域の空気を感じられる 立ち寄りの「カフェリビング」をつくることをテーマとした計画。

OUTLINE

・所在地：群馬県 前橋市
・用途：カフェ / 物販
・規模：内装改修
・担当：レイアウト検討 / マテリアルスタディ / 3D・パース作成 など

NOTE

インターンとしてプロジェクトに参加しながら、素材の質感や光の入り方を 検討するパース制作を通じて、空間の雰囲気づくりに深く関わった。 将来的には、ここでの経験をもとに、より生活者に近い目線での空間づくりを 広げていきたいと考えている。
