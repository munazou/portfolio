広瀬川源平エリア社会実験（Hirosegawa Genpei）

前橋市 / 都市空間デザイン・社会実験 / 2024
Role: Layout Planning / Installation / Video Production

CONCEPT

広瀬川と市街地のあいだにある「源平エリア」で行われた社会実験。 歩行者動線や滞在スペースを再編し、人が自然に集まり、滞在できる川辺のデザインを検証した。 通行量の多い道路と静かな水辺をつなぐことで、まちの中に“もうひとつのリビング”を生み出している。

OUTLINE

・所在地：群馬県 前橋市 広瀬川源平エリア
・内容：社会実験／ファニチャー設置／レイアウト検討／映像制作
・期間：2024年 夏季
・担当：現地調査・設営支援・記録映像

NOTE

現場のレベル差や段差を活かして、川沿いの動線を多層的に構成。 休憩・通行・視線といった行為を重ね合わせることで、単なる通路を“滞在する道”へと変化させている。 記録映像では、実際の人の流れと時間の変化を観察し、都市の“間”に生まれる新しい活動を記録した。
